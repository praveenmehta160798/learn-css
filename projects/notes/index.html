  34. Color Modes

    1. Color Name
      • red
      • green
      • blue
      • orange
      • yellow
      • pink
      • magenta
      • brown
    2. Hexadecimal
      • #ff0000 - red
      • #0000ff - blue
      • #00ff00 - green
    3. RGB(red green blue) - values between 0 to 255
      • rgb(255,0,0) - red
      • rgb(0,250,0) - green
      • rgb(0,0,250) - blue
    4. RGBA(red green blue alpha) - alpha values between 0 to 1 for tranparancy
      • rgb(255,0,0,0.5) - red
      • rgb(0,250,0,0.1) - green
      • rgb(0,0,250,0.5) - blue
    5. HSL(hue saturation lightness) - hue values between 0 to 255, saturation and lightness value between 0 to 100%
      • hsl(0,0%,0%)
      • hsl(120,0%,50%)
      • hsl(240,25%,0%)
    6. HSLA(hue saturation lightness alpha)
      • hsl(0,0%,0%,0.5)
      • hsl(120,0%,50%,0.1)
      • hsl(240,25%,0%,0.9)
